# arduino-kitt-scanner
Knight Rider KITT scanner with speed setting and 15 programs to choose

Adaption for Attiny24:

    int lightPins[] = {4, 3, 2, 1, 0, 8, 9, 10};
    int programPin = 5;
    int speedPin = A6;
